# MSME Vendor Payment Tracking System

Backend API system for managing vendors, purchase orders, and payments for MSMEs.
Built using NestJS, TypeScript, and PostgreSQL.

---

## Tech Stack
- Backend: NestJS (TypeScript)
- Database: PostgreSQL
- ORM: Prisma
- Authentication: JWT
- API Documentation: Postman
- Deployment: Railway / Render

---

## Setup Instructions

### Prerequisites
- Node.js (v18+)
- PostgreSQL
- npm

### Installation
```bash
npm install
